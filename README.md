# Credit_Card_Financial_Report_Dashboard

## Project Overview
This Power BI project provides an interactive analysis of credit card transactions and customer data. The dashboard is designed to help identify revenue trends, customer behavior, and key business performance indicators.

## Dashboard Pages

### 1. Credit Card Transaction Report
- Revenue Analysis
- Transaction Volume
- Interest Earned
- Quarterly Performance
- Revenue by Card Category
- Revenue by Expenditure Type

### 2. Credit Card Customer Report
- Customer Segmentation
- Revenue by Age Group
- Revenue by Income Group
- Revenue by State
- Revenue by Marital Status
- Customer Insights
